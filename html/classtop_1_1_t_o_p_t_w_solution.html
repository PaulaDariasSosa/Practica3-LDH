◆ initSolution()

void top.TOPTWSolution.initSolution ( )

Method to initialize the solution.

The solution is initialized with the depot as the first node of the first route.

The predecessors and successors arrays are initialized with -1.

The routes array is initialized with -1.

The available vehicles are initialized with the total number of vehicles.

The objective function value is initialized with -1.

The waiting time array is initialized with -1.

The position in route array is initialized with -1.
